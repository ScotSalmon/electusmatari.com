Uploader Application

The following is an OOC description.

The Gradient Material Index works by uploading data directly from your game client's cache director to electusmatari.com. To do so, you need to run a client locally, and then just start browsing the market - with the "market history" tab open. The application will see the cache files in the background and upload them automatically. You have to do nothing else.

Installation

To install the Gradient Market Uploader, you need to download the following two files and put them into a directory of your choice (e.g. C:\Program Files\EVE GMU\).

  • CacheUploader.exe
  • CacheUploaderCore.dll

In addition to this, you might need .NET 4.0 from Microsoft:

  • .NET 4.0

The program will update automatically in the future.
